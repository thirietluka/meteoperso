# meteoperso
Code source de l'application MétéoPerso

# Introduction
J’ai effectué un stage de quatre semaines dans une entreprise spécialisée dans la sécurité
des systèmes d’information. L’objectif de ce stage était de découvrir la manière dont on
conçoit et sécurise un petit projet informatique de bout en bout.
Le projet qui m’a été confié s’appelle « Météo perso ». Il consiste à créer une application qui
surveille les ressources de l’ordinateur (processeur, mémoire, disque), stocke les résultats
dans une base de données PostgreSQL, puis affiche ces informations sur une page web
dynamique hébergée par Flask. L’application web utilise une API REST sécurisée en TLS et
présente les résultats sous forme de graphiques.
